# explication:
#   3*x + 4*y + 8 = 0  ⟹  y = -3/4 * x - 2    ⟹  b = -3/4 = ρ σy/σx
#   4*x + 3*y + 7 = 0  ⟹  x = -3/4 * y - 7/4  ⟹  b = -3/4 = ρ σx/σy
#   (-3/4)² = ρ²  ⟹  ρ = ± 3/4
#   b < 0  ⟹  ρ = - 3/4
